Technology & Software Development

I've always been interested in technology since a young age. I started programming when I was around 11 years old, and I've been doing it ever since. I've worked with many different languages and frameworks, but by far my favourites are C and Rust; I also have used LLVM IR, C++, Python, HTML, CSS, and JS. On top of that, I've dabbled in some frameworks such as Astro, Next.js, and React.

If you asked me a few years ago Python would have been my favourite (as it was basically the only language I knew back then), but I've come to love other programming languages since. Especially C - because it doesn't work all the time, but that just it makes it even sweeter when it does.
