Exeme Language

The Exeme Language is a a compiled language that I'm developing, that focuses on simplicity, scalability, and performance. It is currently in development, and is not ready for use. It has been my main project for a while now, and I'm hoping to have at least an alpha version ready by the start of the 2025 academic year.

The compiler is written in C, and compiles to LLVM IR, which is in turn compiled by clang to an executable. Once a beta version has been achieved, the compiler will be re-written in Exeme. The standard library is also written in LLVM IR, by myself.

Bruty

Bruty is a distributed client-server application for brute-forcing YouTube video IDs using Rust. I originally created it for ivocord (a Discord server), but it can be used by anyone. The server is built using shuttle.rs, and so you can host it for free on their platform.

Snek

Snek is a snake game written in Python that runs entirely in your terminal. It is designed to look like a retro game from the early 80s. It currently is relatively very simple, however I am planning to possibly add more features such as levels if I have the time.
